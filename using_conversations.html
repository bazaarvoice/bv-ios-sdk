Conversations

The Curations module is an easy-to-use wrapper around Bazaarvoice's Conversations API.

  Contact Bazaarvoice about setting up Conversations, and to get your apiKeyConversations.

The completed project for this code walkthrough for Swift and Objective-C is in the SDK path /Examples/Conversations. For a more comprehensive demonstration of the Conversations API, please also see the BVSDKDemo project under /Examples/BVSDKDemo.

Before you begin implementation of ths module make sure you have verified your installed as details in the Installation and Configuration page.

Construct the API Request

At this point you should have the BVSDK installed and initialized, so we can now make an API call to the Conversations API

  1. Add the BVDelegate protocol to your view controller

    • Swift
    • Objective-C
    import BVSDK
class ViewController: UIViewController, BVDelegate {
...
    // ViewController.h
#import <BVSDK/BVConversations.h>
@interface ViewController : UIViewController <BVDelegate> {
}

  2. Fill in the BVDelegate:didReceiveResponse:forRequest: function.

    • Swift
    • Objective-C
    func didReceiveResponse(response: [NSObject : AnyObject]!, forRequest request:
AnyObject!) {
  print("Raw Response: ", response)
}
    - (void) didReceiveResponse:(NSDictionary *)response forRequest:(id)request {
    NSLog(@"Raw Response: %@", response);
}

  3. Make an API call. In the viewDidLoad function of your view controller make an API call to get some reviews:

    • Swift
    • Objective-C
    let request = BVGet(type: BVGetTypeReviews)
request.sendRequestWithDelegate(self)

    BVGet *request = [[BVGet alloc] initWithType:BVGetTypeReviews];
[request sendRequestWithDelegate:self];

  4. Finally, run the application and check the console for a successful API call.


  5. Referencing the Conversations API is crucial to getting and submitting data to Bazaarvoice correctly.

    In order the get more familiar with the Conversations module APIs, you should become familiar with the BVGet and BVPost classes which are used to construct all the Conversations API calls. Explore the example projects in the Examples/ folder of the SDK from this repository.
