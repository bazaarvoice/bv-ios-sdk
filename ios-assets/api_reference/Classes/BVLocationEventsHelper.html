gimbalBeaconSighting:date:identifier:name:contextualTier1:contextualTier2:

Record the device receiving a gimbal beacon sighting event. This method has a direct correlation to beaconManager:didReceiveBeaconSighting:
+ (void)gimbalBeaconSighting:(id)RSSI date:(id)date identifier:(id)identifier name:(id)name contextualTier1:(id)tier1 contextualTier2:(id)tier2

Parameters

RSSI

The RSSI (signal strength) of the beacon sighting.

date

The NSDate that the sighting occured

identifier

The GMBLBeacon’s factory ID.

name

The GMBLBeacon’s name, assigned via the Gimbal Manager

contextualTier1

the BV-specified tier 1 interest associated with this sighting. See BVContextualInterests.h

contextualTier2

the BV-specified tier 2 interest associated with this sighting. See BVContextualInterests.h
