Overview

A collection view to show product recommendations in.

This collection view should be used in combination with `BVRecommendationCollectionViewCell` cells.

– loadRequest:completionHandler:errorHandler: 

Load product recommendations based on data fed in from `request`.
- (void)loadRequest:(BVRecommendationsRequest *)request completionHandler:(recommendationsCompletionHandler)completionHandler errorHandler:(recommendationsErrorHandler)errorHandler

Parameters

request

The request parameters to load
completionHandler

Completion handler which returns an array of BVRecommendedProduct product recommendations
errorHandler

Error handler which returns an NSError if the request has failed.
