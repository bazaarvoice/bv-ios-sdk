Overview

Recommendations can be requested with a limit and optional productId and categoryId filtering.

When filtered to a `productId`, the recommendations will be largely related to that product.

When filtered to a `categoryId`, the recommendations will be limited to products in that category.

`limit` is the maximum number of recommendations to load. Suggested: 20. Max: 50.
