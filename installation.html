Installation

This guide shows you how to install and configure the Bazaarvoice SDK (BVSDK) into your project.

Prerequisites

  • Signed Data Usage Amendment, if not already in place.
  • Minimum iOS deployment target of iOS 8.0.
  • Xcode version 7.0 or higher.
  • Your Client ID.
  • API key(s) for the Bazaarvoice product you have enabled.

1 Select Configuration Options

Step 1: Configure your SDK options.
The values you set in this step will generate the necessary script and BVSDK intialization code to get you up and running quickly.

Enter your client ID:
If you are unsure of your client ID or API key values, you can request help through the support portal or your implementation team.

Select One or More Module(s) Enter API Key:
BVCurations
BVConversations
BVAdvertising
BVRecommendations
2 Install the SDK With CocoaPods

Step 2: Configure your Podfile for CocoaPods installation.

Installation with Cocoapods is the recommended approach to including BVSDK in your project. If you don't have Cocoapods installed on your machine, see the Cocoapods installation guide.

Edit your Podfile to include the following:

# Default install: Installs BVConversations and core modules.
pod 'BVSDK'

When you have configured your Podfile, simply run pod install from your project directory to install the BVSDK modules. Then you can open the corresponding .xcworkspace file CocoaPods has created.

Great! The SDK is included in your project. You can add the required imports to your project in the next step.

3 Configue Headers/Imports

Step 3: Add the right header file imports.

Add the right import statements so you can reference the BVSDK modules you are working with.


  • Swift
  • Objective-C / Swift with bridging header
// When using Dynamic Frameworks with Swift (not using Objective-C Bridging header),
// use the folling import in each of your .swift files
import BVSDK

// Import BVSDK.h for all modules, including BVConversations
#import <BVSDK/BVSDK.h>
