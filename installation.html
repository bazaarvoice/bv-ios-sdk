Installation
This guide shows you how to install and import the Bazaarvoice SDK into your project.

Prerequisites

  • Signed Data Usage Amendment, if not already in place
  • Minimum iOS deployment target of iOS 8.0
  • Xcode version 7.0 or higher
  • API key to access Bazaarvoice products

Adding the SDK to your project

Installation with Cocoapods is the recommended approach to including BVSDK in your project. If you don't have Cocoapods installed on your machine, see the Cocoapods installation guide.


Edit your Podfile to include the following:

# Default install: Installs BVConversations and core modules.
pod 'BVSDK/BVConversations'
# Installs BVAdvertising and core modules.
pod 'BVSDK/BVAdvertising'
# Installs BVRecommendations Shopper Advertising API and core modules.
pod 'BVSDK/BVRecommendations'
# Installs BVCurations and core modules.
pod 'BVSDK/BVCurations'

When you have configured your Podfile, simply run pod install from your project directory to install the BVSDK modules. Then you can open the corresponding .xcworkspace file CocoaPods has created.

Great! The SDK is included in your project. You can now add the required imports to your project.

Importing the SDK

Installation with Cocoapods is the recommended approach to including BVSDK in your project. If you don't have Cocoapods installed on your machine, see the Cocoapods installation guide.


  • Swift
  • Objective-C / Swift with bridging header
import BVSDK

// Import BVSDK.h for all modules, including BVConversations
#import <BVSDK/BVSDK.h>
// When using BVRecommendations
#import <BVSDK/BVRecommendations.h>
// When using BVAdvertising
#import <BVSDK/BVAdvertising.h>
// When using BVCurations
#import <BVSDK/BVCurations.h>
// When using BVConversations
#import <BVSDK/BVConversations.h>
