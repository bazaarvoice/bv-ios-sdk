Installation

This guide assumes you have a working knowledge of the Bazaarvoice SDK. If you haven't already done so, consider running through our Get Started and Installation guides.

Add the following lines to your Podfile, if not already there:

{% if page.title == "Advertising" %} 
pod 'BVSDK'
pod 'BVSDK/BVAdvertising'
{% elsif page.title == "Product Recommendations" %} 
pod 'BVSDK'
pod 'BVSDK/BVRecommendations'
{% elsif page.title == "Curations Display" or page.title == "Curations Custom Posts" %} 
pod 'BVSDK'
pod 'BVSDK/BVCurations'
pod 'SDWebImage' # Used to load photos asynchronously
{% else %} 
pod 'BVSDK'
{% endif %}

And install the SDK by running:

pod install
