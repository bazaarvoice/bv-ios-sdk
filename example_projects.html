Example Projects

Product Recommendations, Advertising, Curations, and Conversations all come with example projects, which demonstrate how to properly implement these modules. Specifically:

  • Product Recommendations: Shows how you can fetch and display product recommendations for a user.
  • Advertising: Shows how you can utlize Bazaarvoice's shopper segment data to provide targeting advertising to your customers.
  • Curations: Shows hows to fetch and display curations content beautifully. It also demonstrates how creating and uploading Custom Posts.

  • Conversations has three example applications:

    • BrowsePhotoExample: Illustrates using the Bazaarvoice iOS SDK to query, browse and display ratings and reviews for products.
    • KitchenSink: Covers all the baseline method calls and their responses. Itâ€™s a good way for developers see the all the requests and responses of the SDK out of the box.
    • PhotoUploadExample: Illustrates using the Bazaarvoice iOS SDK to submit a photo-review that includes a user generated photo (camera/gallery), star rating and review text.


Running the examples

For any example project:

  1. Navigate to the project in the Examples/ directory/
  2. Run pod install and open the corresponding .xcworkspace file with Xcode.
  3. Open the AppDelegate file, and set your API keys and clientId in the didFinishLaunchingWithOptions: method.

  4. Select the build target from the project's menu bar:

  5. Run it!
