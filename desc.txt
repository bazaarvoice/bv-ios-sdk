The Bazaarvoice SDK is an iOS static library that provides an easy way to generate REST calls to the Bazaarvoice API.

<h2>SDK Organization:</h2>

Requests:
At a high level, all requests are formed from subclasses of BVBase. Once all the request parameters are in place, set a delegate and call startAsynchRequest: to receive BVDelegate callbacks.

- BVBase
	- BVDisplay*
	- BVSubmission
		- BVSubmission*

Parameters:

- BVParameters
	- BVSubmissionParametersBase
		- BVSubmissionParameters*

Other Classes:

- BVResponse - Encapsulates the response from the server
- BVSettings - Encapsulates application level params for all requests
- BVParametersType - Encapsulates a multi-part request parameter
