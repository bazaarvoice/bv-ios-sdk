FAQ & Support

Frequently Asked Questions


  • Does the BVSDK Use IDFA?

    Yes, the IDFA is used for both the BVAdvertising and BVRecommendations modules. The BVSDK will check if the user has "Limit Ad Tracking" enabled, and respects it if enabled.

    Any applications using the BVAdvertising or BVRecommendations modules will need to submit their applications via iTunes Connect letting Apple know that IDFA is using for targeted advertising.
