Authenticity using device fingerprinting

Bazaarvoice has partenered with iovation, an industry leader in device reputation technology, to ensure authenticity of user generated content. The Bazaarvoice native mobile SDKs for Android and iOS support the ability to include the iovation fingerprint along with Conversations API requests using the BVPost request object.

Please contact your Bazaarvoice support representative to request an evaluation of the iovation integration. Further tehcnical information can be found on the Bazaarvoice developer portal.


iOS Integration

Once you have the iovation.framework integrated into your project and the DevicePrint is started, the integration is very simple. For any BVPost request object created, simply add these lines:

  • Swift
  • Objective-C
let request = BVPost(type:POST_TYPE);
request.fingerPrint = DevicePrint.blackbox();
...
BVPost *request = [[BVPost alloc] initWithType:POST_TYPE];
request.fingerPrint = [DevicePrint blackbox];
...

Sample Application

The Conversations Kitchen Sink application contains iovation hooks that are enabled when you add the iovation.framework to the project. If want to run the Kitchen Sink app with iovation, just follow these steps:

  1. From a terminal navigate to /Examples/Conversations/KitchenSink and run pod install.
  2. Run open BVSample.xcworkspace
  3. Drag and drop the iovation.framework into the BVSampleApp project.
  4. Configure the project with your client id and API key.

That's it! Just run the app and view the BVPost requests in BVSampleAppMainViewController for reference.
