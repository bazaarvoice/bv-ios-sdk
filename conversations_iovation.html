Authenticity using device fingerprinting

Note: Iovation is requied for US customers implementing UGC Content Submission with the Conversations API.

Bazaarvoice has partenered with iovation, an industry leader in device reputation technology, to ensure authenticity in consumer-generated content. Bazaarvoice Mobile SDKs for Android and iOS support the ability to include the iovation fingerprint along with Conversations API requests, using the BVPost request object. Contact your Bazaarvoice support representative to request an evaluation of the iovation integration.

Refer to the Device Fingerprinting spage of the Bazaarvoice Developer Portal for technical details about creating and submitting device fingerprinting information from iovation with consumer-generated content. Refer to the Mobile Device Fingerprinting page of the Developer Portal for mobile-specific details about integrating with iovation.


iOS Integration


Important: Before you start implementing this Mobile SDK module, verify your installation by checking the steps on the Installation and Configuration page.

After you integrate the iovation.framework and the DevicePrint starts, the rest of the integration process is simple. For any BVPost request object created, add the following:

  • Swift
  • Objective-C
let review = ReviewSubmission( ... )    

review.fingerPrint = DevicePrint.blackbox()

// ...add in your other parameters (reviewText, reviewTitle, userNickname, etc.)...
BVReviewSubmission *review = [[BVReviewSubmission alloc] init... ];  

review.fingerPrint = [DevicePrint blackbox];

// ...add in your other parameters (reviewText, reviewTitle, userNickname, etc.)...

Validating Your Iovation Integration

After you install the iovation SDK with the Bazaarvoice Mobile SDK in your application, you can begin to POST Conversations content. After you have sent events, such as the one outlined in the SampleApplication, contact your Bazaarvoice implementation team or authenticity team to verify your integration is set up and working correctly in your production environment.
